### 1. Clone the repository

\`\`\`bash
git clone [your-repo-url]
cd [your-repo-name]

### 2. Install dependencies

\`\`\`bash
npm install
\`\`\`

### 3. Configure Web3Forms

1. Go to [web3forms.com](https://web3forms.com)
2. Enter your email: **ayandajr11@gmail.com**
3. Get your free access key
4. Open `/components/ContactSection.tsx`
5. Replace `YOUR_WEB3FORMS_ACCESS_KEY_HERE` with your actual key (around line 22)

### 4. Run locally

\`\`\`bash
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) in your browser.
